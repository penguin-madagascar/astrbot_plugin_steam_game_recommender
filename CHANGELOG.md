# 更新日志

## 0.7.0 - 2026-07-14

- 推荐候选改为只接受经 Steam 类型确认的游戏本体，并统一处理同作不同版本、游戏库过滤和随机推荐边界。
- 空结果 LLM 兜底改为显式专用模型且默认关闭，同时安全迁移旧配置。
- 调整推荐评分与 Dashboard 配置，完善语言、预算和价格数据不足时的降级行为。

## 0.6.1 - 2026-07-13

- 将库内随机推荐命令改为 `/randomrec`，中文 alias 改为 `/随机推荐`，不再保留旧命令名称。
- 将 README 调整为面向使用者的项目介绍，并用组合示例说明游戏库过滤与区域参数。

## 0.6.0 - 2026-07-13

- 项目和插件 ID 改名为 `astrbot_plugin_steam_game_recommender`，展示名改为“Steam 游戏推荐助手”。
- 删除非 Steam 候选匹配、游戏详情命令和已停用的排序模块；账号绑定与推荐记忆改为 Steam 专用模型。
- 将推荐结果改为 0–100 连续分数，加入对数知名度、贝叶斯口碑、正负参考、游戏库画像、数据完整度和预算调整。
- Steam 语言支持改为支持、不支持、未知三态，并区分简体中文与繁体中文。
- 新增 `-US`、国区、美区、日区等查询级区域；价格只查询指定区域的当前价、历史最低和最近促销。
- 每款游戏独立生成 2–3 句证据化 LLM 理由，最多 5 路并发；单款校验失败时独立降级。
- `/unplayedrec` 改为普通单条消息，仅返回游戏名和玩法、口碑、知名度理由。
- 精简 `/gamerec` 开场白与每款游戏输出，仅保留标题、推荐分、推荐理由、区域价格和 Steam 链接。
