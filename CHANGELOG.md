# 更新日志

## 0.7.0 - 2026-07-17

- 推荐结果会更稳定地排除 DLC、试玩版等非游戏内容，并在核对玩法和价格后选择同一作品中更合适的版本；价格增强会使用准确的 Steam AppID，不同币种不再直接比较，资料较旧或无法确认时也会明确提示。
- 加强个人数据保护：Steam Web API Key 不再出现在请求地址、聊天回复或普通日志中，缓存会自动限制占用，近期推荐记录会在 30 分钟后删除；新增 `/accountunbind` 与 `/解除绑定`，可同时删除绑定和相关个人缓存。
- QQ 官方、Telegram 和 Discord 现在会收到分块文本结果，“换一批”会按会话和用户分别保存；`/randomrec` 最多检查 50 款并跳过单款资料失败。无结果灵感推荐仍默认关闭，仅在 Steam 查询正常但没有合适结果时提供明确标注的灵感。

## 0.6.1 - 2026-07-13

- 将库内随机推荐命令改为 `/randomrec`，中文命令改为 `/随机推荐`，不再保留旧命令名称。
- 将 README 重新整理为用户指南，并补充游戏库过滤与地区参数的组合示例。

## 0.6.0 - 2026-07-13

- 插件包名改为 `astrbot_plugin_steam_game_recommender`，展示名改为“Steam 游戏推荐助手”。
- 推荐范围调整为 Steam 游戏，并停止提供非 Steam 推荐和游戏详情命令。
- 推荐结果改用 0–100 推荐分，并综合玩法、参考游戏、Steam 口碑、关注度、语言、预算和个人游戏库偏好排序。
- 语言支持会区分简体中文、繁体中文和资料未知；查询还支持 `-US`、国区、美区、日区等地区写法。
- 安装价格插件后，可以查看指定地区的当前价、历史最低和最近促销。
- 每款游戏会显示 2–3 句简短理由；个别理由生成失败时不会影响其他结果。
- `/unplayedrec` 改为单条消息，只显示游戏名和简短理由。
- `/gamerec` 输出精简为标题、推荐分、理由、区域价格和 Steam 链接。
